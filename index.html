Tags: Smart Home | Embedded Systems | IoT | Full Stack

Smart Motion Alert System

ECE 50861 - Embedded Systems - November 2024

Tags: IoT | Embedded Systems | Cloud Integration | Automation

Engineered a real-time motion detection and alert system using ESP32 and MPU-6050 accelerometer with ±0.01 g sensitivity. Enabled smartphone control and cloud-based monitoring with sub-2s response time and 97%+ detection reliability.

  • Firmware: MicroPython-based ESP32 logic for real-time acceleration tracking and calibration
  • Connectivity: Real-time event logging and alerting through ThingSpeak and IFTTT
  • Control: Smartphone-triggered configuration and Google Assistant voice commands
  • Performance: Achieved 97%+ accuracy and <2s latency; tuned sensor thresholds to minimize false positives
  • Impact: Demonstrated scalable architecture for low-cost IoT security products
